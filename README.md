# .png への道

## 実行環境の必要条件

- python >= 3.10
- pygame >= 2.1

## ゲームの概要
* 主人公キャラクターcat(.jpeg)が.pngになるために冒険するゲーム
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* wasdでcatを操作
* 敵を倒しつつ、アイテムを拾いながら最上階を目指してください

## ゲームの実装

### 共通基本機能
* ランダムで生成されるマップとランダムで更新できるマップ

### 分担追加機能
* 罠（担当：武沢弘海）：マップのどこかにダメージの異なる罠を設置（透明）
* キッチンタイマー機能（担当：ぷしみ）：制限時間以内に調理が完了しなかった場合に，豪華豚が脱走する機能
* 調理機能（担当：ぶしみ）：調理器具をキー押下により選択し，豪華豚を調理する機能
- 敵クラス（担当：フィッシャー）：マップ内の部屋に１対の敵をランダムな場所に描画
* プレイヤーのパラメーターを取得するクラスの作成(森本)
* レベルを参照しステータスを決定する関数の作成(森本)
* 経験値が一定を超えた場合レベルアップする関数の作成(森本)

### メモ
* クラス内の変数は，すべて，「get_変数名」という名前のメソッドを介してアクセスするように設計してある
* すべてのクラスに関係する関数は，クラスの外で定義してある
