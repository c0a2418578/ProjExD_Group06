# .png への道

## 実行環境の必要条件

- python >= 3.10
- pygame >= 2.1

## ゲームの概要

- 主人公キャラクター cat(.jpeg)が.png になるために冒険するゲーム
- 参考 URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方

- wasd で cat を操作
- 敵を倒しつつ、アイテムを拾いながら最上階を目指してください

## ゲームの実装

### 共通基本機能

- ランダムで生成されるマップとランダムで更新できるマップ

### 分担追加機能

- 罠クラス（担当：武沢）：マップ内にランダムで罠を生成（透明）

### メモ

- クラス内の変数は，すべて，「get\_変数名」という名前のメソッドを介してアクセスするように設計してある
- すべてのクラスに関係する関数は，クラスの外で定義してある
